Magnum Opus error messages

The following error messages may be generated by Magnum Opus.   During data import, where possible, Magnum Opus generates an error message, discards the current line from the input file, and continues.  Errors for which this is possible are called recoverable errors.  The remaining errors are called non-recoverable errors.   On a non-recoverable error, the current action is aborted.

Note that the identifier-item format allows little data checking.  Any value in an appropriate column will be accepted as an identifier or item.  The item-list format allows no data checking.

All data import error messages specify the file and line on which the error was encountered.  Where relevant, the column is also specified.  For example, consider the following names file and data file.

Names file ‘test.nam’:

a: a,b
b: continuous
c: 0,1

Data file ‘test.data:

a,1,0
b,0 1

The following error message will be generated, indicating the line and column on which the error has occurred (in this case, a missing comma).

Error on line 2 column 2 of test.data: Missing or incorrect delimiter following value for attribute 'b'.
    Encountered '1' when expecting ','.

Error messages relating to all files

WARNING: the (estimated) data set size exceeds the capacity of this computer's physical memory.
Processing may be significantly slowed by use of virtual memory.

Magnum Opus estimates that it will require more memory to process the current data set than the amount of physical memory installed in the host computer.  If this estimate is correct then processing is likely to be significantly slowed by the use of virtual memory.  This message is displayed to provide the user with the opportunity to reduce the data set size.  This can be achieved by sampling.

Where possible Magnum Opus augments this message with a suggested data sample percentage that will avoid this problem.  Due to the difficulty of accurately estimating the amount of memory that will be required these estimates may prove inaccurate and should be taken as an approximate guide only.

This is a warning message only.
 

Cannot open names file 'filename'

An error has occurred when trying to open the names file.   Non-recoverable.
 

Cannot open data file 'filename'

An error has occurred when trying to open the data file.   Non-recoverable.
 

Cannot open holdout data file 'filename'

An error has occurred when trying to open the holdout data file.   Non-recoverable.
 

Data capacity exceeded. Remaining data discarded.

The data file contains more than the system limit of 2,147,483,647 cases.  Recoverable.
 

Test data capacity exceeded.  Remaining data discarded.

The test data file contains more than the system limit of 2,147,483,647 cases.   Recoverable.
 

Maximum number of items exceeded, no more than 2,147,483,647 items allowed

The data contain more than the system limit of 2,147,483,647 items.   Non-recoverable.
 

No items imported

Data import was completed but no items were loaded, possibly due to data import errors.   Non-recoverable.
 

No cases loaded

Data import was completed but no cases were loaded, possibly due to data import errors.   Non-recoverable.
 

Error messages relating to the names files

Attribute declared with no values. Data import aborted

An attribute declaration has been encountered that lists an attribute name but no values. Non-recoverable.
 

Cannot use '?' as an attribute value name. Data import aborted

'?' has been declared as an attribute value.  '?' is reserved for use as a missing value and cannot be declared as a value in the names file.  Non-recoverable.
 

Data import aborted due to invalid attribute declaration 'line'

A line has been encountered that does not start with the comment character (|) and does not conform to any of the supported attribute declaration formats.  Non-recoverable.
 

Data import aborted due to declaration of multiple attributes with the same name 'name'

The same attribute name has been declared twice.  Non-recoverable.
 

Data import aborted due to duplicate declaration of value 'name' for attribute att

The same attribute value has been declared twice for a single attribute.  Each possible value for an attribute must have a distinct name.  Each value must listed once in the declaration of the attribute.  Non-recoverable.
 

Data import aborted due to bad cut specification.  Encountered 'c' when expecting '<' or '<='

An attribute has been declared using sub-ranges, but one of the sub-range values is followed by unexpected text.  Non-recoverable.
 

Data import aborted due to invalid cut specification.  Numeric range range is empty

An attribute has been declared using sub-ranges, but one of the sub-ranges cannot contain any values.  For example, an attribute declared as x < 1 < 0 would create three sub-ranges, x<1, 1<=x<0, and x>=0.  The range 1<=x<0 cannot contain values.  Non-recoverable.
 

Data import aborted due to missing cut value following '<' or '<='.  Encountered 'c' when expecting a number in declaration for attribute 'name'

An attribute has been declared using sub-ranges, but non-numeric text has been encountered where a numeric sub-range cut value is expected.  Non-recoverable.
 

Data import aborted due to missing attribute name

A line that should start with the name of an attribute starts with a colon (:).  Non-recoverable.
 

Data import aborted due to invalid generalization declaration.
Cannot create a generalization for non categorical attribute 'att'

An attempt has been made to define a generalization for an attribute that is not categorical.  Non-recoverable.
 

Data import aborted due to invalid generalization declaration.
Cannot create a generalization for dynamic categorical attribute 'att'

An attempt has been made to define a generalization for an attribute that was declared with the keyword categorical rather than a list of allowed values.  You can only define generalizations of categorical attributes that are defined with a list of allowed values. Non-recoverable.
 

Maximum number of attribute values exceeded, no more than 2,147,483,647 attribute values allowed

The system limit of 2,147,483,647 attribute values has been exceeded.  Non-recoverable.
 

Missing string terminator (") after string

A line contains an odd number of quote characters.  Either the start or end quote character might have been left off a string, or a quote character might have been used inadvertently within a name.  Non-recoverable.
 

Unexpected character 'c' following string "string"

The character following a string does not conform to the required format.  Non-recoverable.
 

Error messages relating to data files

'name' is not a value for attribute att

The column for a categorical attribute does not contain an acceptable value.  Only the missing value (?) or one of the values that appears in the attribute declaration for the attribute should appear in a column for a categorical attribute.  One possible cause for this error is a missing column delimiter following a missing value.  Recoverable.
 

Missing or incorrectly specified numeric value.
Encountered 'c' when expecting a value for numeric attribute att.

The column for a numeric attribute does not contain an acceptable value.  Only the missing value (?) or a number should appear in a column for a numeric attribute.  Possible causes for this error include the omission of a column delimiter following an empty field for the attribute, or the inclusion of a superfluous column delimiter following the column for a prior attribute.  Recoverable.
 

Missing or incorrect delimiter following value for attribute 'att'.
Encountered 'c' when expecting 'c'.

A column contains an acceptable entry but is not followed by the delimiter that has been specified.  Possible causes for this error include a typographical error in the attribute value, inconsistent use of delimiters within the data file, selection of the wrong character as delimiter, or omission of the delimiter in the data file.  Recoverable.
 

More values than attributes

A line contains more attribute values than the number of attributes declared in the names file.  One possible cause for this error is the inclusion of a superfluous column delimiter.  One common cause of this is when the name of an attribute value includes the delimiter, for example, if a comma is part of a name.  This can be resolved by either changing the delimiter or by putting quote marks (") around the name, for example "clothing, outerwear".  Recoverable.
 

Fewer values than attributes

The line contains fewer attribute values than the number of attributes declared in the names file.  Every line of the data file must contain one column for each declared attribute.   All columns must be separated by the delimiter that is selected.  The content of each column must be the value of the corresponding attribute for the case that the line describes.  Recoverable.
 

Unexpected character 'c' following missing value indicator '?'

The column for a numeric attribute contains a question mark (?) followed by other text.  Only the missing value (?) or a number should appear in a column for a numeric attribute.  One possible cause for this error is the omission of a column delimiter following a missing value.  Recoverable.
 

Error messages relating to identifier-item files

Missing transaction identifier

The column for an identifier is empty.  Recoverable.
 

Missing transaction item

The column for an item is empty.  Recoverable.
 

Too few columns

Either the transaction identifier or the transaction item is missing.  Recoverable.
 

WARNING: There are very few items per case.
Perhaps the wrong delimiter was specified!

This warning is issued if at the end of data import the average number of items per case is very low.  This often occurs if the wrong delimiter is specified.  A look in the list of items will readily identify whether the items are correct.  Recoverable.

 
Error messages during rule or itemset discovery

Unable to evaluate unsound filter because the size of the search space exceeds system capacity

The size of the search space exceeds the system’s capacity to calculate an unsound filter.  This error can avoided by decreasing the maximum size of the rules or itemsets, decreasing the number of items or attribute values allowed, or using holdout evaluation instead of the unsound filter.
 

Out of memory while evaluating rules against holdout data.
Holdout evaluation is incomplete.

The system was not able to complete holdout evaluation due out running out of memory.  This may be avoided by decreasing the maximum number of rules or itemsets allowed, or decreasing the maximum size of the rules or itemsets.
 

Out of memory while checking for partial sets.
Check is incomplete.

The check for partial itemsets could not be completed and some partial itemsets may remain in the output results.
 

Cannot evaluate sets containing more than 32 elements
No such set has been subjected to holdout evaluation

The system is unable to perform holdout evaluation of itemsets containing more than 32 items.
